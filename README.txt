		** FREE BEER **

El objetivo del proyecto es elaborar recetas comunitarias de cerveza. 
Hacer cerveza artesana y pretender ganar dinero con ello es más complicado de lo que parece. Daré las claves para aproximar un precio con el que no se pierda dinero.
Yo voy a dar el valor inicial, pero me gustaría que las iteraciones fuesen comunitarias.

En este repositorio colgaré etiquetas, un Calc de LibreOffice para adaptar las recetas al volumen de cerveza que queráis cocer y saber cuánto os cuesta. Podéis enviar todos los commits que queráis. Yo intentaré ser un cocinero más y un árbitro. Empezamos?
